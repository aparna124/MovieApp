# MovieApp
 

## Installing Expo-CLI

```
$ npm install --global expo-cli
```

Installing the node modules.

If using multiple node versions,

```
$ nvm use v14.17.0
```
Install dependencies

```
$ npm install
```
## How to use API Key

Replace <API_KEY> in POPULAR_MOVIE_API and SEARCH_API at file api.js with your private key.


## Run Project

Start Bundling
```
$ expo start
```
or

```
$ npm start
```

## Test Project 

```
$ npm test
```
